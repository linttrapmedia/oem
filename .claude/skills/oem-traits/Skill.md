# Trait Library

Traits are reusable functions that give abilities and behaviors to HTML elements. They are
registered in the `Template()` config and accessed via the `trait` proxy. Each trait receives
the element as its first argument, followed by configuration parameters.

## Focus

_Focuses an HTML element based on conditions_



---

## InputEvent

_Handles input-related events on form elements_



---

## TextContent

_Sets the text content of an HTML element_



---

## useAttributeTrait

_Adds an attribute to an HTML element_



---

## Style

_Applies CSS styles to an HTML element_



---

## InputValue

_Binds a value to an input or textarea element_



---

## Event

_Attaches event listeners to an HTML element_



---

## InnerHTML

_Sets the innerHTML of an element with reactive children_



---

## ClassName

_Sets the class name of an HTML element_



---

## StyleOnEvent

_Applies CSS styles to an element on a specific event_
