## Quick Start

### How it works

OEM enforces a syntax convention which aims to support the **locality of behavior** pattern where markup, style, logic and behavior are colocated using TypeScript. This creates a seamless human-AI collaboration experience where the human can easily understand and modify the code generated by the AI, and the AI can generate code that is easy for the human to understand and modify.

### Installation

```bash
npm install @linttrap/oem
```

### Agent Skills

```bash
npx skills add https://github.com/linttrapmedia/oem/tree/main/docs/agent-skills
```
