# OEM

UI, on command.

[**Human Docs at oem.js.org**](https://oem.js.org)

OEM is an agent-first UI framework and toolkit complete with solutions for state management, templating, styling including a full component library and a carefully composed agent-skill definitions for unlimited extensiblity. It's minimal, flexible foundation makes it ideal for crafting your own reactive, component-based UI layer that an AI agent can generate with ease and a human can understand at a glance.
